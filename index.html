    • OTiS - Optimised Template System is a tool that i created in order to help me at my work as a Customer Support Agent. My work had a lot of repetitive cases and we had templates that we had to copy from excel files where those were stored and fill them up. This process was time consuming and i came up with the idea of embedding these templates into a notation tool in order to make it easier and faster to handle customer's concerns and not only as this is a very usual process in any department of a company of filling up a template and sending it somewere.

    I participated to an International inovation contest at my company and won 1st place.

    The skills acquired in this project are:

    • DOM manipulation using JavaScript and jQuery
    • ES6 standarts
    • RegExp for the template-ing feature
    • LocalStorage to save the preferences and templates locally
    • Excel files manipulation using xlsx library
